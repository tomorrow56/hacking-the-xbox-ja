# 付録A　ハッキング機器の入手先

ハードウェアハッキングを始める際の心理的な障壁のひとつが、「ハッキングに必要な道具が手に入らない」という思い込みだ。Radio Shackでも部品や工具は少し揃っているが、値段は高いし、今の小型化された部品に対応できる工具は少ない。この付録では、豊富な品揃えをリーズナブルな価格で提供しているコンポーネントおよびツールのベンダーをいくつか紹介する。また、本書のプロジェクトに必要な基本工具のJamecoの注文番号も掲載した。これらの番号をJamecoのセキュアな注文サイトに直接入力するだけで、すぐに始められる。

## ホビイスト向けベンダー

## 機材注文フォーム

このセクションでは、Jamecoで購入できる厳選した工具リストを紹介する。「基本」注文フォームには、Xboxを開けるのに必要なビットとドライバ類、および基本的なはんだ付け工具がすべて含まれている。「上級」注文フォームには、あると便利だが、予算が限られている場合は必要になってから個別に購入することも検討できる高めの工具が含まれている。

これらの注文フォームにある工具に加えて、第I部で説明するプロジェクトにはいくつかの小さな部品が必要になる。必要な部品のリストは、それぞれの章の導入部を参照してほしい。

26ピースSAEホビーツールセットには、T-10、T-15、T-20トルクスビットを含む、Xboxの分解に必要なビット、ドライバ、スクリュードライバがすべて揃っている。注文フォームに1/32"コニカルベベルチップを追加したのは、付属のはんだごてに付いているチップが1/8"という巨大なチゼルチップだからだ。1/8"チップは、小さな国をはんだ付けするくらいには使えるかもしれないが、ファインピッチの部品全体にはんだショートを量産する元凶にもなる。コニカルベベルチップとチゼルチップには素早い熱伝達を可能にする平らな面があるため、先細りのコニカルシャープチップより使いやすい。はんだごてのチップクリーナーとロジンペーストフラックスも含めた。少し値が張るが、作業がぐっと楽になる——信じてほしい。

### 上級注文フォーム

以下の表は、あると便利だが、ハッキングをたっぷりやるつもりがある場合にのみ購入を検討すべき高めの工具の一覧だ。

- はんだごてスタンドとスポンジは、少しでもはんだ付けをするつもりなら必須だ。スタンドに熱いこてを安全に置いておけるので、やけどや火災のリスクを減らせる。
- 1/32"セミチゼルチップではんだ付けしてみてほしい。コニカルベベルチップより気に入るかもしれない。
- 静電防止リストストラップは、特に静電気が発生しやすい服や靴を着用する傾向がある人には常に手元に置いておくべきものだ。
- SMT除去キットは、小さなSMT部品を取り外すときに重宝する。
- マルチメータは、いつでも使える万能な計測・診断ツールで、自宅や寮でさまざまな場面で活躍する。多くのモデルがあるが、ここにリストしたものは最も安い価格で最多機能を提供している。実際に試してみたが、よく使える。
- ここに掲載されているワイヤーストリッパーとクリンプ工具は、電源ユニット交換プロジェクトに非常に役立つ。クリンプ工具はJamecoではなくDigi-Keyから購入する。少し高めだが、10ドル以下の工具と違い、成形クリンプダイが付いていて高品質なはんだ不要の圧着が可能だ。
